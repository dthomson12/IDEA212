Sketch Library

A Library is basically a Sketch file/document containing Symbols that can be used globally (as opposed to just locally). You can share a Library by either storing it on a Local Server, by using a Cloud Service such as Dropbox, or even via a Version Control system like GitHub or Abstract. As a result, you and your teammates can have access to the same Libraries from one central location.

Create a Library

  • Add a new sketch file
  • Save the file as ClientName_Library (for your assignment, save it as FirstNameLastName_Library.

Bring a library (or as many as you want) into Sketch

Manually add it as a Sketch Library via Preferences > Libraries > Add Library.

Naming Conventions are IMPORTANT

There are many different file structures / naming conventions that you can use:

  • Atomic Design
  • OOCSS
  • SMACSS
  • Conventions that are easy to apply for working in Sketch is BEM.

BEM (Block Element Modifier)

  • EG Button/Primary/Enabled
  • EG Button/Primary/Disabled
  • EG Button/Secondary/Enabled
  • EG Button/Secondary/Disabled

Adding Colors

Add boxes of the same size and style them using your primary and secondary colors. In the right panel under "Appearance" add a new layer style.

eg. Naming Convention: Palette/Primary/Black

Adding Typography

Add text layers for H1, H2, H3, H4, H5, link, body etc. In the right panel under "Appearance" add a new layer style.

eg. Naming Convention: Text/H1
